# 📊 CodeAlpha - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this task is to perform Exploratory Data Analysis (EDA) on an employee dataset to understand its structure, identify patterns, detect outliers, and generate meaningful insights using Python.

---

## 🎯 Objectives

* Load and inspect the dataset
* Analyze data structure and statistics
* Check for missing values
* Perform correlation analysis
* Detect outliers
* Generate business insights

---

## 🛠️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

Employee Dataset containing:

* EmployeeID
* Name
* Department
* Age
* Salary

---

## 📋 Steps Performed

1. Imported required libraries
2. Loaded the dataset
3. Explored data using `head()`, `info()`, and `describe()`
4. Checked missing values
5. Selected numerical columns
6. Created a correlation heatmap
7. Detected outliers using a boxplot
8. Interpreted the results

---

## 📈 Key Insights

* No missing values were found.
* Age and Salary show a moderate relationship.
* Salary distribution varies among employees.
* No major outliers were observed.

---

## 📸 Output

* Dataset Summary
* Correlation Heatmap
* Salary Boxplot

---

This project improved my understanding of data exploration, statistical analysis, and data preprocessing techniques.
